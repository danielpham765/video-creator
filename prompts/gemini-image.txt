PHASE 2 — HARD BOUNDARY CROP (VERY IMPORTANT)

After cleaning dialogue, you MUST crop the image.

CRITICAL: HOW TO DETECT THE TRUE ARTWORK AREA

You MUST ignore visual meaning and use STRUCTURAL evidence.

An area belongs to artwork ONLY if it contains:

• ink outlines
• drawing strokes
• contour lines
• object edge lines
• shadow edges attached to objects or characters

This is a pixel structure rule, not a scene understanding rule.

Color, gradient, fog, or brightness DO NOT count.

WHAT IS MARGIN (IMPORTANT)

The following are ALWAYS margin:

• flat white space
• gradient sky with no outlines
• empty gray area
• watermark zones
• compression noise
• blank webtoon padding

Even if it looks like background.

If no ink outline exists → it is margin.

OPEN WEBTOON RULE (THIS FIXES YOUR ISSUE)

Many webtoon panels have NO bottom border.

In this case:

The artwork ends at the last pixel row containing any ink outline or shadow edge connected to an object or character.

Everything below that row MUST be deleted.

This is mandatory.

EXACT CROPPING ALGORITHM (FOLLOW STRICTLY)

You must scan image edges.

Bottom boundary (most important)

Starting from the bottom of the image:

Move upward pixel row by pixel row.

Find the FIRST row that contains ANY drawing stroke or outline pixel.

That row is the TRUE artwork boundary.

Delete every pixel below it.

You are NOT allowed to keep extra padding.

Top boundary

From the top of the image:

Move downward until a drawing outline appears.

Crop everything above it.

Left and Right

From both sides:

Move inward until an outline or object edge appears.

Crop outside area.

Safety padding

After boundary detection:

Add only 6 pixels padding.

No more.

Keeping large margins is an error.

STRICT FAILURE CONDITION

If large blank space remains at top or bottom, the task is considered failed.

You must trim tightly.
